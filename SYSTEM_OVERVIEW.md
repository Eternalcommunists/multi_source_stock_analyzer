## 系统功能概述

### 1. 基础分析功能
- **技术分析**: 趋势、动量、支撑压力、技术指标（MA、RSI、MACD等）
- **基本面分析**: 估值、财务健康度、增长潜力
- **风险评估**: 市场、估值、流动性、技术面风险
- **综合评分**: 1-10分制投资评分系统

### 2. 高级分析功能（新增）
- **财报分析**: 深度财务报表分析与健康度评估
- **机构持股分析**: 机构投资者行为追踪与信心评估
- **做多做空分析**: 市场情绪与做空兴趣度分析
- **主流机构观点**: 摩根士丹利等主流投行观点整合
- **综合评分系统**: 基础分析与高级分析融合的评分

## 系统架构

### 后端模块
- `core_analyzer.py`: 核心分析器
- `advanced_analyzer.py`: 高级分析器（财报、机构、做空分析）
- `integrated_analyzer.py`: 集成分析器（整合基础与高级分析）
- `main_enhanced.py`: 增强版主程序入口
- `html_generator.py`: HTML报告生成器

### 技术栈
- **Python**: 核心分析引擎
- **akshare**: 金融数据获取
- **yfinance**: 国际金融数据接口
- **pandas/numpy**: 数据处理
- **Flask/FastAPI**: API服务

## 快速开始

### 分析单个股票
```bash
cd D:/Codex/multi_source_stock_analyzer
python backend/main_enhanced.py --symbol 000001 --output-format all
```

### 批量分析股票
```bash
# 分析多个股票
for symbol in 000001 600036 000858; do
    python backend/main_enhanced.py --symbol $symbol --output-format enhanced
done
```

### 使用编程接口
```python
from backend.main_enhanced import quick_analyze

# 快速分析
result, html_report, json_report = quick_analyze('000001', ['akshare'])
```
